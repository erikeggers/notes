# Arrays
A type that is an ordered list of values.

## Creating an array with literal syntax
- `var theArray = ['hay', 123]`

## Getting values out of an array
- `0` is the first index
- `var hay = theArray[0]`
- `var num = theArray[1]`

## Changing an array
- `theArray[1] = 321`
- `theArray[2] = "cool"`

## Array length
- `var length = theArray.length`
